# 🌐 Java Web Scraper using JSoup

A Java-based Web Scraper developed as part of the **SkillCraft Technology Software Development Internship (Task 04)**. This application extracts product information from an online e-commerce website using the **JSoup** library and stores the extracted data in a structured **CSV** file.

---

## 📌 Project Overview

The main objective of this project is to demonstrate web scraping using Java by collecting product details from an online website. The scraper connects to the website, extracts useful information such as product name, price, rating, availability, and product URL, and saves the data into a CSV file for further analysis.

---

## 🚀 Features

- Extracts Product Name
- Extracts Product Price
- Extracts Product Rating
- Extracts Product Availability
- Extracts Product URL
- Stores extracted data in CSV format
- Uses JSoup for HTML parsing
- Simple and clean Java implementation
- Console output for verification

## ⚙️ How to Run

### Clone Repository

```bash
git clone https://github.com/shabeerasheik/SCT_SD_4.git
```

### Open Project

Open the project using **Visual Studio Code**.

### Add JSoup Library

Download the JSoup JAR file and place it inside the **lib** folder.

### Compile & Run

Compile and execute the Java program.

The scraper will:

- Connect to the website
- Extract product details
- Save data into `products.csv`
